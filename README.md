# Installation
### Clone the repository:
```git clone  https://github.com/es10176/Mocha.git```
### Install dependencies
```pip install -r requirements.txt```
# Usage
### 🔹 Pass URL as an argument
```python scraper.py https://example.com```

### 🔹 Interactive mode
```python Mocha.py```
